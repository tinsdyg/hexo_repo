# Redis
## 1. 安装直接使用命令 pkg install redis 即可
## 2. 安装完成后 redis-server 会失败报内核错误, 需要修改运行配置 redis.conf  (在 usr/etc 中)
## 3. 把末尾那行 ignore-warnings ARM64-COW-BUG 的注释去掉
## 4. 为了使 Redis 能在后台运行, 需要在配置文件中 把 daemonize 设置为 yes
